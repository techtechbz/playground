1. 5 * 2 (整数×整数=整数)
2. 0.5 * 2 (小数×整数=整数)

4. 8.4 * 2.1 (小数×小数=小数)
5. 5.0 * 2.0 (小数×小数=整数)
6. -4 * 8 (負の数×正の数=負の数)

8. -4 * -8 (負の数×負の数=正の数)

## c++

```c++
#include <iostream>
using namespace std;

int main() {
  cout << 5 * 2 << endl;
  cout << 0.5 * 2 << endl;
  cout << 0.5 * 3 << endl;
  cout << 8.4 * 2.1 << endl;
  cout << 5.0 * 2.0 << endl;
  cout << -4 * 8 << endl;
  cout << 4 * -8 << endl;
  cout << -4 * -8 << endl;
}
```
